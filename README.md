# Flappy Bird HTML5 Game

Ein modernisiertes Flappy Bird Spiel, das in reinem HTML5, CSS3 und JavaScript programmiert wurde.

## Features

- **Dynamische Geschwindigkeit**: Die Röhren bewegen sich langsamer am Anfang und werden mit steigendem Score immer schneller
- **Verbesserte Steuerung**: Klarere Sprunghöhe und langsamer Fall
- **Optimierte Kollisionserkennung**: Fehlerfreies Spielverhalten
- **Zusätzliche Grafikverbesserungen**: Bessere Wolken-Rendering und visuelle Details

## Spielsteuerung

- **Leertaste** oder **Klick** zum Fliegen
- **Leertaste** zum Neustart nach Game Over

## Installation

1. Laden Sie die `flappy-birds-fixed.html`-Datei herunter
2. Öffnen Sie sie in einem modernen Webbrowser
3. Viel Spaß beim Spielen!

## Entwicklung

Das Spiel ist vollständig in einer HTML-Datei implementiert, mit integriertem CSS und JavaScript.
